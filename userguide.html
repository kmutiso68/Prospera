Prospera - User Installation Guide

This guide will help you install Prospera on your ARM-based Mac.

Prospera

System Requirements

  • Operating System: macOS
  • Architecture: ARM-based Mac (Apple Silicon)
  • Terminal: Required for installation

Installation Process

Once downloaded, head to where you installed it; if not sure where it is look for the searchbar in finder and type in Prospera.pkg (if you don't see it, try installing again); if you try to double click the pkg, apple's built in OS security tool will display a dialog reporting cannot verify that app does not have malware.

Important: Do not select "Move to Bin"

Installation Steps:

1. Right click on the pkg file
2. Click on "Get Info"
3. Look for the "Where" section and copy the text you find there, starting from the word "Users" all the way till the last word in the where section
4. Paste this command somewhere (i.e., Notes or somewhere else) - we will refer to it later, this is known as your file path
5. If you know how to open Terminal go ahead and open it, if not: press and hold briefly (start by holding down the command key then pressing the spacebar) the command key and the space bar together, you should see the word Spotlight Search
6. Type in "Terminal" and press Enter
7. Welcome to Terminal, paste the following command on the terminal, do not press enter yet, if you did go back to step 5:
sudo xattr -rd com.apple.quarantine
8. Press spacebar (it should be the pasted command, then one space)
9. Paste the file path you saved somewhere (the one from notes or wherever you saved/pasted it), your terminal should now have sudo xattr -rd com.apple.quarantine <path>, now press Enter
10. Congratulations! We have bypassed Apple's OS security tool that blocks installation of this pkg. Do not worry, we have only done this for this app, the security feature itself has not been disabled
11. Do not worry, your Mac will remember this choice so you won't need to run this command again
12. Close Terminal
13. You can now double click the pkg to install Prospera

Despite all this we are still not done, as the app will not run directly. The developer of this app is in the process of obtaining their developer certificate from Apple which unfortunately affects the running of this app. Apologies for the inconvenience, this will be fixed in future iterations.

14. Right click on the app (if you don't know where it is, it should be in your Applications folder in Finder)
15. Click on "Show Package Contents"
16. Click on Contents/open Contents folder, click on MacOS/open MacOS folder
17. You should see a file called Fundingplatform (a unix executable file don't worry about what this means), this is the application
18. Finally, open the file (double click it)

If it runs, congratulations on successfully installing Prospera and thank you for your patience!

If it doesn't run, please check the FAQs section below.

FAQs

Why didn't the app run?

Something must have gone wrong in either the installation process or, storage of the file, or maybe (less likely) something more subtle.

I'm still having issues with the installation. What should I do?

Please contact our support team at support@prospera.com with details about the issue you're experiencing.

Will future updates require the same installation process?

No, once we obtain our developer certificate from Apple, future updates will follow the standard installation process.
